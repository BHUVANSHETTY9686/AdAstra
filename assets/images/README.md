# Instructions

Place your images in this folder to use them in the website.

## Recommended Files:

### Hero Section:
- `hero-bg.jpg` - Main background image (1920x1080 or larger)

### Featured Games:
- `project-nebula.jpg` (1600x900)
- `void-walker.jpg` (1600x900)
- `starlight-chronicles.jpg` (1600x900)

### Gallery:
- `gallery-1.jpg` through `gallery-6.jpg` (1200x800)

### Other:
- `logo.png` - Optional custom logo
- `studio-photo.jpg` - Team photo or studio shot

## After adding images:

1. Update the paths in `adastra.html`
2. Commit: `git add . && git commit -m "Add images"`
3. Push: `git push origin main`
